# House model test

WebVR scene built with A-Frame.

### Quick-start

```bash
npm install
npm start
```

Then launch the site from your favourite browser:

[__http://localhost:3000/__](http://localhost:3000/)

If you wish to serve the site from a different port:

    PORT=8000 npm start
